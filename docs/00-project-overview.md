# Project Overview

## Purpose
Tsubasa AI Project は、社内AI基盤を整備し、業務効率化・ナレッジ蓄積・経営判断支援を実現するための基盤プロジェクト。

## Scope
初期フェーズでは以下を対象とする。
- 社内AI相談室
- 会議議事録の整理
- 社内ドキュメント検索
- タイヤ事業データの整理
- 営業支援

## Rollout Phases
1. Phase 1: 社内AI相談室
2. Phase 2: 社内知識検索
3. Phase 3: 一部業務自動化
